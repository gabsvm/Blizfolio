# BizFolio - Company Management System

A production-grade frontend application for managing company profiles and product portfolios.

## Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite (simulated)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Forms**: React Hook Form + Zod
- **Routing**: React Router (HashRouter)

## Features

- **Authentication**: Login/Logout with mock session persistence.
- **Company Profile**: Detailed form with validation for business identity, location, and fiscal data.
- **Product Management**: Container/Folder based organization system.
- **Dashboard**: Real-time mock statistics and activity.
- **Theme**: Light/Dark mode support.

## Project Structure

- `/components`: Reusable UI and business components.
- `/services`: Mock API layer simulating backend delay and DB.
- `/store`: Global state management.
- `/pages`: Route views.
